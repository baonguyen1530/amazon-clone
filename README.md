# Amazon Frontend Clone

This project is a frontend clone of the Amazon website, developed using HTML, CSS, and JavaScript. It replicates the look and feel of Amazon's main interface and includes basic interactivity using JavaScript.

## Project Overview
This is a simple frontend clone of Amazon's homepage and a few inner pages. It focuses on layout and design elements of the original site while keeping the code lightweight and easy to understand.

## Features
- Responsive design for desktop and mobile.
- Basic navigation through HTML links.
- Basic product listings and categories.
- Interactive search bar (with JavaScript).
- Dropdowns and hover effects.

## Technologies Used
- **HTML5**: For the structure and layout of the web pages.
- **CSS3**: For styling and layout responsiveness.
- **JavaScript**: For simple interactivity such as the search bar and dropdown menus.

## Setup and Installation

### Prerequisites
- A modern web browser (Chrome, Firefox, etc.).
- A text editor (VSCode, Sublime Text, etc.) if you want to modify the code.

### Steps
1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/amazon-frontend-clone.git
    ```
2. Navigate to the project directory:
    ```bash
    cd amazon-frontend-clone
    ```
3. Open `index.html`, right click and click on "Open with Live Server". (You should install Live Server extensions on VSCode)
